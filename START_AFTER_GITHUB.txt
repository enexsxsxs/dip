Команды для запуска Laravel-проекта после скачивания с GitHub (PowerShell)

1) Откройте терминал в папке проекта:
cd C:\Users\Lenchik\Desktop\dip\laravel

2) Установите зависимости и выполните базовую настройку:
composer install
copy .env.example .env
php artisan key:generate

3) Откройте файл .env и укажите данные вашей базы:
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=your_db_name
DB_USERNAME=root
DB_PASSWORD=

4) Заполните базу данных (выберите один вариант):

ВАРИАНТ A (у меня есть скачанная БД .sql):
# Импортируйте .sql в свою базу через phpMyAdmin -> Импорт
# После импорта migrate --seed обычно не нужен

ВАРИАНТ B (если разворачиваю структуру Laravel с нуля):
php artisan migrate --seed

5) После импорта БД вручную вставьте папку с файлами:
# Скопировать папку:
каталог_оборудования
# Вставить в:
storage\app\public\
# Должно получиться:
storage\app\public\каталог_оборудования

6) Создайте (или обновите) ссылку на storage:
php artisan storage:link

7) Очистите кеш проекта:
php artisan optimize:clear

8) Запустите проект:
php artisan serve

9) Если в проекте есть фронтенд (Vite/NPM), запустите отдельно:
npm install
npm run dev

10) Проверка в браузере:
- Сайт открывается по адресу из php artisan serve
- Картинки/документы доступны из папки каталог_оборудования
- При необходимости путь в шаблонах: asset('storage/каталог_оборудования/имя_файла.ext')
